# l-garage-by-Lukeee-1.0.0
L-Garage skripty jsou plně standalone řešení bez potřeby frameworku. Nabízí jednoduchou instalaci, přehlednou konfiguraci a podporu Discord rolí pro řízení přístupu. Jsou optimalizované pro výkon a snadno upravitelné. Pro správné fungování je nutné mít nainstalovaný ox_lib. Nastavení zvládneš během pár minut. Stačí přidat do server.cfg. ✔
